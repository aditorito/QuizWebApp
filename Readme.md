## Run Locally

Clone the project

```bash
  git clone https://github.com/aditorito/QuizWebApp
```

Go to the project directory 

```bash
  cd QuizApp
```

Install dependencies(Do it both files backend and frontend)

```bash
  npm install
```

Run this command to start project

```bash
  npm run dev
```
